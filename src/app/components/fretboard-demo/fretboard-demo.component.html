Features Demonstrated:

  • Renders 6 strings (EADGBE) with 12 frets by default
  • Color-coded notes (root notes in red, scale notes in blue)
  • Fret markers at standard positions (3, 5, 7, 9, 12)
  • Hover tooltips with note details
  • Support for alternative tunings (Drop D)
  • Responsive design that adapts to container size
  • ARIA labels for accessibility
